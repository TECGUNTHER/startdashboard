Welcome to StartDashboard

One-time setup for this browser. About 90 seconds.

Step 1 · Create a GitHub Personal Access Token

StartDashboard stores your dashboard data in a private GitHub gist on your account. We need a token to read and write that gist.

  1. Open github.com/settings/tokens (the "Fine-grained tokens" page).
  2. Click Generate new token. Name it StartDashboard.
  3. Set expiration to 1 year (or longer).
  4. Under Account permissions → Gists, set access to Read and write.
  5. Click Generate token, then copy the token. It only shows once.

Or use a classic token from this preconfigured page (only gist scope needed).
