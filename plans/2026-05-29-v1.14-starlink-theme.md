# Build Plan: StartDashboard v1.14 — Starlink background theme

**Date:** 2026-05-29
**Status:** Approved
**Plan author:** Design Agent (via Orchestrator)
**Approved by:** Tom (chose: background preset + matched, overridable electric-blue accent)

## The Idea, In Plain English

Add a "Starlink" option to the background picker. Picking it on a persona applies a cinematic deep-space backdrop matching the Starlink website, and auto-switches that persona's accent to a Starlink-style electric blue (overridable). Inter font + glass buttons + white-on-dark already match, so this completes the look. CSS-only, no images, per-persona, synced.

## What It Will Do
- New "Starlink" thumbnail in Settings → background presets, beside the existing 9.
- Selecting it sets `persona.background = 'starlink'` → cinematic near-black deep space (scattered stars, faint diagonal Milky Way band, subtle blue/teal nebula gradients).
- Selecting it also sets the accent color input to a Starlink electric blue (≈`#4F9CFF`, Build Agent finalizes to match screenshots), live-previewed and saved on Save; fully overridable afterward.
- Per-persona, persists to gist, syncs across devices. Dark mode is its home; light mode keeps its existing light-surface override (no conflict).

## What It Will NOT Do
- No global font/button overhaul (Inter + glass buttons already match).
- No image files (pure CSS).
- No Starlink light-mode variant.
- Doesn't lock the accent (auto-applied on selection only; overridable).

## Files to Be Changed
| File | What |
|------|------|
| `projects/personal-dashboard/index.html` | `.bg-starlink` CSS preset; `BG_PRESETS` entry `{id:'starlink', name:'Starlink', accent:'#4F9CFF'}` (optional accent field added to preset shape); accent auto-apply when a preset with an `accent` is clicked in the bg-preset grid |

## Approach
- `.bg-starlink`: near-black base (#05060A) + layered `radial-gradient` star dots (varied positions/sizes) + faint diagonal `linear-gradient` Milky Way band + 2–3 large soft low-opacity nebula `radial-gradient`s (deep blue/teal/faint purple). Start from the existing `.bg-deep-space` / `.bg-aurora` patterns; tune cinematic + restrained to match the Starlink screenshots.
- `BG_PRESETS`: add the Starlink entry with an optional `accent` field.
- Accent auto-apply: in the bg-preset thumbnail click handler (openSettingsModal / bg-preset-grid logic), if the clicked preset has `accent`, set the accent color input value to it (live preview) so Save persists it to `p.accent`. Presets without `accent` leave accent untouched (current behavior).
- Picker grid + `applyPersonaTheme` already render dynamically from BG_PRESETS / swap `.bg-<id>` — no change needed there.

## Dependencies
None new. No fonts, no images, no services.
